# Online Voting System

A secure web-based Online Voting System developed as an academic project.

## Tech Stack
- Java
- JSP & Servlets
- JDBC
- MySQL
- HTML, CSS, JavaScript
- Apache Tomcat

## Features
- Voter registration and login
- Unique voter authentication
- Prevention of multiple voting
- Secure vote storage
- Admin panel for candidate management
- Real-time result monitoring

## How to Run
1. Import project into Eclipse
2. Configure Apache Tomcat
3. Create MySQL database
4. Run on server and open in browser
